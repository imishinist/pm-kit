# Project Setup Interview

## Purpose

Enrich the `project.yaml` generated by `pm-kit create` through an interactive conversation with the user.

## Procedure

Ask the user the following questions in order, and reflect their answers in `project.yaml`.

### 1. Project Overview

- What is the project's purpose and goal?
- How would you describe it in one sentence? (reflects into description)

### 2. Jira Configuration

- Do you use Jira?
- If yes:
  - Jira URL (e.g., `https://company.atlassian.net`)
  - Project key (e.g., `RNW`)
  - Board ID (can be found in the board URL)
  - Board type: Scrum or Kanban?

### 3. Slack Configuration

- Are there Slack channels related to this project?
- Please provide channel names (multiple allowed)

### 4. Confluence Configuration

- Do you use Confluence?
- If yes:
  - Confluence URL
  - Space key (e.g., `BILLING`)

### 5. Related Repositories

- Are there Git repositories related to this project?
- Please provide repository names and URLs (multiple allowed)

### 6. Meetings

- Do you have regular meetings?
- How are meeting notes managed? (Google Docs / manual)

## Applying Results

Write the interview results into `project.yaml`. Leave unanswered fields commented out.

Once complete, guide the user to configure authentication credentials in `.envrc`.
